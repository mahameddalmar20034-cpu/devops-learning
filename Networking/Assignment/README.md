The aim of the assignment was to spin up an ec2  web server that was linked to a domain name.The server used nginx which the code to build this was integrated in the user data alongside the creation of the server.The server  had an allocated IPv4 adress which was used to match it to the domain that was bought from Cloudflare.This meant that searching up that specific domain name allowed the browser to connect to the ec2 server's specific IP.A CNAME record was also created to open up browser domain options like "www." alongside the actual domain name of "modalmar.co.uk".The server allowed traffic from port 80 using the HTTP protocol so that it could be searched up alongside port 22 using the SSH protocol.I used the SSH protocol to troubleshoot nginx downloading server issues.When the website loads up it shows the classic "Welcome to nginx!" web page.
